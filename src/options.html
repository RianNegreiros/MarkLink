MarkLink Options

Keyboard Shortcut

MarkLink comes with a default keyboard shortcut: Alt+M

You can customize this shortcut by following these steps:

  1. Go to chrome://extensions in your browser
  2. Scroll down and click on "Keyboard Shortcuts" at the bottom of the page
  3. Find "MarkLink" in the list
  4. Click in the box next to "Copy current page as Markdown link"
  5. Press your desired key combination
  6. Click "OK" to save
Note: Some key combinations might already be in use by Chrome or other extensions and won't be available.

Other Ways to Use MarkLink

In addition to the keyboard shortcut, you can also:

  • Click the MarkLink icon in your browser toolbar
  • Right-click on any page and select "Copy as Markdown Link"
